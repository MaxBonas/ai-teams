### 🔴 CRÍTICO

| ID | Área | Problema | Archivo:línea | Estado |
|---|---|---|---|---|
| C1 | Memory | Sin límite de tamaño — JSONL crece indefinidamente | `memory.py:51-55` | **CORREGIDO** |
| C2 | Quality gates | Sin timeout — workflow se bloquea forever si Reviewer desaparece | `orchestrator.py:1202` | **CORREGIDO** |
| C3 | Security | `redact_text()` nunca se aplica a contenido de memoria — API keys en disco | `memory.py:remember()` | **CORREGIDO** |
| C4 | Syncthing | Sin `.stignore` — puede sincronizar venv/, runtime/, .env entre máquinas | (ausente) | **CORREGIDO** |
